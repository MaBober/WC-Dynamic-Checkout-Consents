Dynamic Checkout Consents lets you add customizable consents to the WooCommerce checkout page. Define required or optional consents.

== Description ==  
**Dynamic Checkout Consents for WooCommerce** allows store owners to add flexible consent checkboxes on the WooCommerce checkout page. You can:  
- Define multiple consents with custom text  
- Make consents required or optional  
- Assign consents to specific products or categories  
- Store accepted consents in order metadata  
- Display consents in WooCommerce admin order details  
- Use basic HTML formatting in consent text (`<b>`, `<i>`, `<a>`, `<br>`)

This plugin is useful for GDPR compliance, marketing consents, and other checkout agreements.

== Installation ==  
1. Upload the plugin folder to the `/wp-content/plugins/` directory.  
2. Activate the plugin through the ‘Plugins’ menu in WordPress.  
3. Go to **WooCommerce → Settings → Accounts & Privacy** to configure consents.  

== Frequently Asked Questions ==  
= Can I assign a consent to a specific product? =  
Yes! You can choose whether a consent applies to all products, specific products, or entire product categories.  

= Can customers see which consents they agreed to? =  
The accepted consents are stored in the order details and visible to admins.  

= Does this plugin support HTML in consent text? =  
Yes, you can use simple formatting like `<b>`, `<i>`, and `<a href="">`.
